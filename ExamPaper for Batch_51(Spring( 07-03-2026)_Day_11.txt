Ques 1 — Loan Interest Calculation System

Develop a Spring Core application using Strategy Pattern to calculate loan interest based on loan type.

Requirements

1. Interface
InterestStrategy

Method:

double calculateInterest(double amount);

2. Implement classes

HomeLoanInterest

Interest Rate = 7%

CarLoanInterest

Interest Rate = 9%

3. Service class

LoanService

Inject InterestStrategy

Method:

void processLoan(double amount)

4. Output
Display calculated interest.

Example Output
---------------
Loan Type: Car Loan
Loan Amount: 200000
Interest Amount: 18000


Ques-2 — Food Ordering System

Develop a Spring Core application to implement Factory Design Pattern for a Food Ordering System.

Requirements

Create an interface Food

Method

void prepareFood();

Implement two classes:
Pizza
Burger

Create a FoodFactory class

Return object based on input type (pizza / burger)

Print food preparation message.

Example Output
Case 1

Enter food type: pizza
Preparing Pizza...
Pizza is ready to serve

Case 2

Enter food type: burger
Preparing Burger...
Burger is ready to serve

Ques-3 — Travel Booking System

Develop a Spring Core application using @Qualifier annotation for a Travel Booking System.

Requirements

Create an interface BookingService

Method

void bookTicket();

Implement four classes:
FlightBooking
TrainBooking
BusBooking
HotelBooking

Create a TravelManager class

Inject dependency using @Autowired and @Qualifier

Book service using selected booking type.

Example Output
---------------
Booking Type: Flight
Processing flight booking
Booking completed successfully
