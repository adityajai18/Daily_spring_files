Task 1 — Student Details Binding System

Classes Involved:

Student

StudentRunner (Main/Runner class)

Student Class Fields:

id

name

course

fee

Responsibility:

Bind properties using prefix student

Runner Class Responsibility:

Fetch Student object

Display student details

Output:

---- Student Details ----
ID      : 101
Name    : Rahat
Course  : B.Tech
Fee     : 75000

Task 4 — Bank Account Validation System

Classes Involved:

BankAccount

BankRunner

BankAccount Class Fields:

accountNumber

holderName

balance

Responsibility:

Bind properties

Validate balance > 0

Runner Class Responsibility:

Display account or error
Valid Output:

---- Bank Account Details ----
Account No : 1234567890
Name       : Rahman Khan
Balance    : 5000

Invalid Output:

Error: Balance must be greater than 0
